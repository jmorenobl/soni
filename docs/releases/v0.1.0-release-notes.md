# Soni Framework v0.1.0 - MVP Release

We're excited to announce the first alpha release of Soni Framework!

## What's New

Soni Framework v0.1.0 is the first MVP release, providing:

- **Automatic NLU Optimization** - Uses DSPy's MIPROv2 to optimize dialogue understanding
- **Stateful Dialogue Management** - Built on LangGraph for reliable conversation flows
- **YAML-Based Configuration** - Declarative DSL for defining dialogue flows
- **Async-First Architecture** - High-performance async/await throughout
- **REST API** - FastAPI server with `/chat` and `/health` endpoints
- **CLI Tools** - `soni optimize` and `soni server` commands

## Limitations

This is an MVP release with known limitations:
- Linear flows only (branching in v0.3.0)
- SqliteSaver instead of AsyncSqliteSaver (v0.2.0)
- Validators referenced by name only (v0.4.0)

See [CHANGELOG.md](CHANGELOG.md) for complete details.
