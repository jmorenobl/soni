## Migration from v0.1.0

### Breaking Changes

- **AsyncSqliteSaver:** All checkpointing now uses async. Update any custom checkpointers.
- **Streaming:** New optional streaming endpoint. Non-streaming endpoint unchanged.

### Upgrade Guide

```bash
# Update dependencies
uv sync

# No code changes required for basic usage
# Streaming is opt-in via new endpoint
```
